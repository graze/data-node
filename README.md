# data-node

<img align="right" src="http://media1.giphy.com/media/pGmqerZ1NTdZe/giphy.gif" />

Data Nodes for use with [Data\Flow](https://github.com/graze/data-flow)

## Install

Via Composer

``` bash
$ composer require graze/data-node
```

## Testing

```bash
$ make test
```
